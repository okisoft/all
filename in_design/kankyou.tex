\section{コード規約}
ファイル命名規則は原則Railsの命名規則に従う

\begin{itemize}
\item コード命名規則
  \begin{itemize}
  \item クラス名，モジュール名は原則UpperCamelCase
  \item 変数名，メソッド名は原則snake\_case
  \item 定数名は全て大文字で'\_'で区切る
\end{itemize}
\item コーディングスタイル
\begin{itemize}
  \item インデントには半角スペース 2 文字を使用する
  \item 文字コードはUTF-8とする
  \item 改行コードにはLFを使用する
  \end{itemize}
\end{itemize}
